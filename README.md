# 33기 2차 프로젝트 - wantUs 백엔드



## 개요 

- 내용 : wanted 웹페이지 클론코딩 프로젝트
- 기간 : 2022.06.07 ~ 06.17
- 인원 : 총 6명 (프론트엔드: 4명, 백엔드: 2명)
- 구현 기능
  - 소셜 로그인(카카오, 네이버, 구글)
  - 채용 공고 API (리스트, 상세)
  - 좋아요(Follow) API
  - 채용 결과 API

## 사용한 stack & tool
- Django, Python, MySQL, AWS(RDS, EC2, S3)
- github, slack, trello
